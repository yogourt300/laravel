# ESN Manager

App de ticketing Laravel 11 pour gérer projets, tickets et temps passé.

## Rôles

- **Admin** : tout voir, tout gérer (users, projets, tickets)
- **Collaborateur** : ses projets assignés, créer/modifier des tickets, saisir des heures
- **Client** : ses projets, consulter les tickets, valider ou refuser les tickets `a_valider`
